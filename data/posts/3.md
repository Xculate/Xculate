## 一、问题：跨设备传输的日常痛点

### 典型场景
- 电脑上复制了一段代码，想在手机上粘贴，需要打开微信发送"文件传输助手"
- 手机上收到一份PDF，想传到电脑编辑，要经历"保存到文件→发送→电脑下载"三步
- 截图了一张重要信息，想在电脑上使用，得先保存再发送

### 现有方案的缺陷

| 方案 | 问题 |
|------|------|
| 微信/QQ | 需要登录、压缩画质、有大小限制、步骤繁琐 |
| 网盘 | 上传下载等待、需要网络、操作层级深 |
| 数据线 | 找线、连接、点信任、找文件夹、无法无线 |
| 邮件 | 附件大小限制、收发延迟 |
| AirDrop | Windows不支持 |

**核心需求：无线、无感、秒级响应**

## 二、原理：它是怎么工作的？

### 2.1 整体架构

| 步骤 | iPhone端 | Windows端 |
|:----:|----------|-----------|
| ① | 敲后盖触发快捷指令 | - |
| ② | 快捷指令运行JavaScript | - |
| ③ | 发起HTTP请求 | → |
| ④ | - | Python服务器处理请求 |
| ⑤ | 接收响应 | ← |
| ⑥ | - | 读写剪贴板/文件 |
| ⑦ | 更新手机剪贴板/保存文件 | - |

**数据流说明**

| 方向 | 流程 |
|:----:|------|
| **剪贴板交换** | 手机先读取电脑剪贴板 → 再发送自己剪贴板 → 将电脑内容复制到手机 |
| **电脑→手机文件** | 电脑Ctrl+7选文件 → 手机请求下载 → 接收并保存 |
| **手机→电脑文件** | 手机选择文件 → 上传 → 电脑保存到桌面 |

### 2.2 三个关键组件

| 组件 | 作用 | 为什么选它 |
|------|------|-----------|
| **Windows Python服务器** | 接收请求、读写电脑剪贴板、暂存文件 | Python标准库即可实现，无需安装复杂框架 |
| **快捷指令 + JavaScript** | 发起HTTP请求、处理返回数据 | 系统自带，可被敲后盖触发 |
| **Scriptable** | 提供JavaScript运行环境、访问剪贴板和文件 | 免费、功能强大、与快捷指令无缝集成 |

### 2.3 数据流详解

#### 剪贴板交换（双向）
```
手机敲后盖触发
    ↓
快捷指令运行JavaScript（内联脚本）
    ↓
JavaScript中：
    1. 先 GET /read  → 获取电脑剪贴板内容
    2. 再 POST /write → 发送手机剪贴板内容
    3. 将电脑内容返回给快捷指令
    ↓
快捷指令将返回内容复制到手机剪贴板
    ↓
完成交换
```

#### 文件传输
```
电脑→手机：
    Windows按 Ctrl+7 选择文件 → 服务器暂存（Base64编码）
    → 手机运行"下载文件"快捷指令
    → JavaScript发起 GET /file/check
    → 接收文件数据 → 解码保存到手机

手机→电脑：
    手机运行"上传文件"快捷指令
    → 快捷指令让用户选择文件
    → JavaScript发起 POST /file/upload
    → 电脑接收并保存到桌面
```

### 2.4 为什么用内联JavaScript而不是Scriptable脚本？

快捷指令中有两种方式运行JavaScript：

| 方式 | 说明 | 优缺点 |
|------|------|--------|
| **内联JavaScript** | 代码直接写在快捷指令里 | 无需管理多个脚本、一键修改、适合简单逻辑 |
| Scriptable脚本 | 在Scriptable App中创建独立脚本 | 功能更全、代码可复用、适合复杂逻辑 |

**本方案选择内联JavaScript**，因为剪贴板交换逻辑简单，写在快捷指令里更便于维护和调试。

### 3.3 创建服务器脚本
新建文件 `server.py`，复制以下代码：

```python
import os
import time
import threading
import base64
import json
import socket
from http.server import HTTPServer, BaseHTTPRequestHandler
import pyperclip
import tkinter as tk
from tkinter import filedialog
import keyboard

# ============================================
# 统一服务器 - 剪贴板同步 + 文件传输
# ============================================

class UnifiedServer:
    def __init__(self, port=8080):
        self.port = port
        self.temp_folder = os.path.join(os.path.expanduser("~/Desktop"), "文件传输临时")
        self.temp_file = os.path.join(self.temp_folder, "temp_file.dat")
        self.meta_file = os.path.join(self.temp_folder, "meta.json")
        
        # 剪贴板同步数据（存储文本）
        self.clipboard_data = ""
        self.clipboard_lock = threading.Lock()
        
        # 文件传输数据
        self.pending_file = None
        self.pending_meta = None
        self.pending_file_path = None
        
        os.makedirs(self.temp_folder, exist_ok=True)
        self.clean_temp()
        
    def clean_temp(self):
        if os.path.exists(self.temp_file):
            os.remove(self.temp_file)
        if os.path.exists(self.meta_file):
            os.remove(self.meta_file)
        self.pending_file = None
        self.pending_meta = None
        self.pending_file_path = None
    
    def select_file_for_transfer(self):
        """打开文件选择窗口，选择要传输的文件"""
        root = tk.Tk()
        root.withdraw()
        root.attributes('-topmost', True)
        
        file_path = filedialog.askopenfilename(
            title="选择要传输到手机的文件",
            initialdir=os.path.expanduser("~/Desktop")
        )
        
        root.destroy()
        
        if file_path:
            print(f"📁 用户选择文件: {file_path}")
            self.save_file_to_temp(file_path)
            return True
        else:
            print("❌ 未选择文件")
            return False
    
    def save_file_to_temp(self, file_path):
        try:
            with open(file_path, 'rb') as f:
                file_data = f.read()
            
            if len(file_data) > 100 * 1024 * 1024:
                print(f"⚠️ 文件太大 ({len(file_data)} bytes)，跳过")
                return
            
            encoded = base64.b64encode(file_data).decode('utf-8')
            meta = {
                "name": os.path.basename(file_path),
                "size": len(file_data),
                "type": "file",
                "timestamp": time.time()
            }
            with open(self.temp_file, 'w', encoding='utf-8') as f:
                f.write(encoded)
            with open(self.meta_file, 'w', encoding='utf-8') as f:
                json.dump(meta, f)
            self.pending_file = encoded
            self.pending_meta = meta
            self.pending_file_path = file_path
            print(f"✅ 文件已准备: {meta['name']} ({meta['size']} bytes)")
        except Exception as e:
            print(f"❌ 保存文件失败: {e}")
    
    # ========== 剪贴板同步 API ==========
    def monitor_clipboard(self):
        """监控剪贴板文本变化"""
        print("📋 开始监控剪贴板...")
        last_text = ""
        
        while True:
            try:
                current = pyperclip.paste()
                
                if current and current != last_text:
                    # 只处理文本，不处理文件路径
                    if not os.path.isfile(current):
                        print(f"📝 检测到文本复制: {len(current)} 字符")
                        with self.clipboard_lock:
                            self.clipboard_data = current
                        last_text = current
                
                time.sleep(0.5)
                
            except Exception as e:
                print(f"监控错误: {e}")
                time.sleep(1)
    
    def get_clipboard(self):
        with self.clipboard_lock:
            content = self.clipboard_data
            self.clipboard_data = ""
            if content:
                print(f"📤 返回剪贴板内容: {len(content)} 字符")
            return content
    
    def set_clipboard(self, content):
        with self.clipboard_lock:
            self.clipboard_data = content
        pyperclip.copy(content)
        print(f"📥 收到手机剪贴板: {len(content)} 字符")
    
    def check_clipboard(self):
        with self.clipboard_lock:
            return len(self.clipboard_data) > 0
    
    # ========== 文件传输 API ==========
    def get_pending_file(self):
        if self.pending_file and self.pending_meta:
            print(f"📤 返回待传输文件: {self.pending_meta['name']}")
            return {"status": "success", "data": self.pending_file, "meta": self.pending_meta}
        return {"status": "empty"}
    
    def receive_file(self, encoded_data, meta):
        try:
            file_data = base64.b64decode(encoded_data)
            desktop = os.path.expanduser("~/Desktop")
            save_path = os.path.join(desktop, meta.get("name", "received_file"))
            counter = 1
            original_path = save_path
            while os.path.exists(save_path):
                name, ext = os.path.splitext(original_path)
                save_path = f"{name}_{counter}{ext}"
                counter += 1
            with open(save_path, 'wb') as f:
                f.write(file_data)
            pyperclip.copy(save_path)
            print(f"📥 收到文件: {save_path}")
            return {"status": "success", "path": save_path, "size": len(file_data)}
        except Exception as e:
            return {"status": "error", "error": str(e)}
    
    def clear_pending(self):
        if self.pending_file:
            self.clean_temp()
            return {"status": "cleared"}
        return {"status": "no_pending"}


class HTTPHandler(BaseHTTPRequestHandler):
    def do_OPTIONS(self):
        self.send_response(200)
        self.send_header('Access-Control-Allow-Origin', '*')
        self.send_header('Access-Control-Allow-Methods', 'GET, POST, OPTIONS')
        self.send_header('Access-Control-Allow-Headers', 'Content-Type')
        self.send_header('Content-Length', '0')
        self.end_headers()
    
    def do_GET(self):
        try:
            path = self.path.split('?')[0]
            
            if path == '/check':
                has_content = self.server.server.check_clipboard()
                self.send_json({"has_content": has_content})
            elif path == '/read':
                content = self.server.server.get_clipboard()
                self.send_text(content)
            elif path == '/file/check':
                result = self.server.server.get_pending_file()
                self.send_json(result)
            elif path == '/file/clear':
                result = self.server.server.clear_pending()
                self.send_json(result)
            else:
                self.send_error(404)
        except Exception as e:
            print(f"GET 错误: {e}")
            self.send_error(500)
    
    def do_POST(self):
        try:
            path = self.path.split('?')[0]
            
            if path == '/write':
                length = int(self.headers.get('Content-Length', 0))
                data = self.rfile.read(length)
                try:
                    request = json.loads(data)
                    content = request.get('content', '')
                except:
                    content = data.decode('utf-8')
                self.server.server.set_clipboard(content)
                self.send_json({"status": "success"})
            elif path == '/file/upload':
                length = int(self.headers.get('Content-Length', 0))
                data = self.rfile.read(length)
                request = json.loads(data)
                result = self.server.server.receive_file(
                    request.get('data', ''),
                    request.get('meta', {})
                )
                self.send_json(result)
            else:
                self.send_error(404)
        except Exception as e:
            print(f"POST 错误: {e}")
            self.send_error(500)
    
    def send_json(self, data):
        try:
            json_str = json.dumps(data)
            json_bytes = json_str.encode('utf-8')
            self.send_response(200)
            self.send_header('Content-Type', 'application/json')
            self.send_header('Access-Control-Allow-Origin', '*')
            self.send_header('Content-Length', str(len(json_bytes)))
            self.end_headers()
            self.wfile.write(json_bytes)
        except:
            pass
    
    def send_text(self, text):
        try:
            text_bytes = text.encode('utf-8')
            self.send_response(200)
            self.send_header('Content-Type', 'text/plain; charset=utf-8')
            self.send_header('Access-Control-Allow-Origin', '*')
            self.send_header('Content-Length', str(len(text_bytes)))
            self.end_headers()
            self.wfile.write(text_bytes)
        except:
            pass
    
    def log_message(self, format, *args):
        pass


def get_ip():
    try:
        s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
        s.connect(('8.8.8.8', 80))
        ip = s.getsockname()[0]
        s.close()
        return ip
    except:
        return "127.0.0.1"


def hotkey_listener(server):
    """监听快捷键 ctrl+7 打开文件选择"""
    print("⌨️ 快捷键: ctrl+7 - 选择文件传输")
    while True:
        try:
            if keyboard.is_pressed('ctrl+7'):
                print("\n🎯 快捷键触发！")
                server.select_file_for_transfer()
                time.sleep(0.5)
            time.sleep(0.1)
        except:
            time.sleep(0.1)


def main():
    print("=" * 60)
    print("🚀 统一服务器 - 剪贴板同步 + 文件传输")
    print("=" * 60)
    
    server = UnifiedServer(port=8080)
    
    # 启动剪贴板监控线程（只监控文本）
    monitor_thread = threading.Thread(target=server.monitor_clipboard, daemon=True)
    monitor_thread.start()
    
    # 启动快捷键监听线程
    hotkey_thread = threading.Thread(target=hotkey_listener, args=(server,), daemon=True)
    hotkey_thread.start()
    
    # 启动 HTTP 服务器
    http_server = HTTPServer(('0.0.0.0', 8080), HTTPHandler)
    http_server.server = server
    
    ip = get_ip()
    print(f"📡 电脑IP: {ip}")
    print(f"🔌 端口: 8080")
    print("=" * 60)
    print("📋 剪贴板同步:")
    print(f"   - 复制文本 → 自动准备")
    print(f"   - GET  /read  → 读取")
    print(f"   - POST /write → 写入")
    print("=" * 60)
    print("📁 文件传输:")
    print(f"   - ctrl+7 → 手动选择文件")
    print(f"   - GET  /file/check → 检查文件")
    print(f"   - POST /file/upload → 接收文件")
    print("=" * 60)
    print("按 Ctrl+C 停止")
    print("=" * 60)
    
    try:
        http_server.serve_forever()
    except KeyboardInterrupt:
        print("\n⏹️ 服务器已停止")
        http_server.shutdown()


if __name__ == "__main__":
    main()
```

## 五、使用效果

### 5.1 剪贴板同步

| 场景 | 操作 | 结果 |
|------|------|------|
| 电脑→手机 | 电脑复制文字，敲两下手机背面 | 手机剪贴板=电脑文字 |
| 手机→电脑 | 手机复制文字，敲两下背面 | 电脑剪贴板=手机文字 |
| 两边都有内容 | 敲两下背面 | 交换，各得对方内容 |

### 5.2 文件传输

| 方向 | 操作 |
|------|------|
| 电脑→手机 | 电脑按 `Ctrl+7` 选文件 → 手机运行"下载文件" |
| 手机→电脑 | 手机运行"上传文件" → 选择文件 |

### 5.3 响应速度
- 剪贴板交换：< 0.5秒
- 文件传输：1-5秒/MB

## 六、故障排除

| 问题 | 解决方法 |
|------|---------|
| 手机连不上电脑 | 1. 检查同一WiFi<br>2. 确认电脑IP正确<br>3. 检查防火墙 |
| 敲后盖没反应 | 设置→辅助功能→触控→轻点背面→检查是否选中 |
| JavaScript报错 | 检查IP地址是否正确填写 |
| 文件传输失败 | 检查文件大小（建议<100MB） |

## 七、总结

这套方案的核心理念是：**iPhone敲后盖触发 → 快捷指令运行内联JavaScript → HTTP请求Windows服务器 → 双向同步剪贴板和文件**

优势：
- **无感**：敲后盖即触发，不打断当前操作
- **无线**：纯局域网通信，不依赖互联网
- **极速**：文字交换<0.5秒
- **零成本**：仅需Python和免费App
